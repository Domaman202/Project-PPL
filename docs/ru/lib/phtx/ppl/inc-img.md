## inc-img
Читает __картинку__ из `файла`.<br>
***Может встроить картинку в код программы.***

### Применение

1. `(inc-img path)`<br>
`path` - _путь до файла_.

### Примеры

```pihta
(use-ctx pht phtx/ppl
    (app-fn
        (println (inc-img "res/0.jpg"))))
```

***Требует наличие файла `0.jpg` в папке `res` исполняемого модуля.***
